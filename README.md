
# Retail Inventory Analytics Project

## 📊 Overview
This project analyzes retail inventory data using Python.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## 📈 Analysis Performed
- Category-wise sales analysis
- Stock management analysis
- Supplier performance analysis
- Top-selling products identification

## 💡 Key Insights
- Certain categories dominate sales
- Some products require immediate restocking
- Supplier dependency exists
- Stock imbalance identified

## 📁 Files Included
- Notebook (.ipynb)
- Dataset (.xlsx)
- Report (.pdf)
- Dashboard screenshots
